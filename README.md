# MA-HAD-AR-RASHID-AL-ISLAMI
মা’হাদ আর-রশিদ আল-ইসলামী একটি আধুনিক দ্বীনি শিক্ষা প্রতিষ্ঠান, যেখানে হিফজুল কোরআন, নাজেরা ও জেনারেল শিক্ষার সমন্বয়ে ছাত্রীদের আদর্শ হাফেজা ও সুশিক্ষিত নাগরিক হিসেবে গড়ে তোলা হয়। অভিজ্ঞ শিক্ষকমণ্ডলীর তত্ত্বাবধানে আবাসিক ও অনাবাসিক সুবিধাসহ নিরাপদ ও ইসলামিক পরিবেশে মানসম্মত শিক্ষা প্রদান করা হয়।
